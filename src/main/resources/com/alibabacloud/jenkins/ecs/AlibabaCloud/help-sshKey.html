Alibaba Cloud SSH key pair is a safe and convenient way of login authentication. It consists of a public key and a private key. It only supports Linux instances.

The private key is in unencrypted PEM (Privacy-Enhanced Mail) encoded PKCS#8 format. The private key of an ECS key pair starts with "-----BEGIN RSA PRIVATE KEY-----" .
It is recommended to set the credential ID to the name of the key created on the Alibaba Cloud console.
